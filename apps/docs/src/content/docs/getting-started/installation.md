---
title: Installation
description: Install Minions Applications packages
---

## TypeScript / Node.js

```bash
npm install @minions-applications/sdk minions-sdk
```

## Python

```bash
pip install minions-applications
```

## CLI

```bash
npm install -g @minions-applications/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
