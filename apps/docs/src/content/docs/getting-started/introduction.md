# Minions Applications

Application records, submission bundles, platform receipts, and status tracking

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-applications/sdk`) — Core library for Node.js
- **Python SDK** (`minions-applications`) — Python bindings
- **CLI Tool** (`@minions-applications/cli`) — Command-line interface
- **Web App** — Interactive playground at [applications.minions.wtf](https://applications.minions.wtf)
